! Note about loopbacks: on an ABR, these get placed with the lowest area number the ABR is a part of.

router ospf 1
area 3 stub
network 10.0.3.0 0.0.0.255 area 3
network 10.1.1.0 0.0.0.255 area 3
network 172.16.0.5 0.0.0.0 area 3
network 172.16.0.9 0.0.0.0 area 3
network 172.16.0.10 0.0.0.0 area 3
exit

ipv6 router ospf 1
exit
